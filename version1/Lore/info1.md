# SISTEMA DE ATRIBUTOS – PROPOSTA COMPLETA

Vou dividir em 4 camadas:

1. Atributos Primários
2. Atributos Secundários
3. Recursos
4. Estados Espirituais (temático amazônico)

---

# ATRIBUTOS PRIMÁRIOS (Base do Personagem)

Esses definem combate e testes.

| Atributo            | Função                         |
| ------------------  | ------------------------------ |
| **Força**           | Dano físico, armas pesadas     |
| **Agilidade**       | Esquiva, iniciativa            |
| **Espírito**        | Poder espiritual/magia         |
| **Vitalidade**      | Vida máxima                    |
| **Sorte**           | Eventos aleatórios e críticos  |

---

# Versão Temática Amazônica (mesma lógica, nomes adaptados)

| Nome Alternativo   | Equivalente |
| ------------------ | ----------- |
| Bravura            | Força       |
| Destreza           | Agilidade   |
| Conexão Espiritual | Espírito    |
| Resistência        | Vitalidade  |

# ATRIBUTOS SECUNDÁRIOS (Derivados)

Calculados automaticamente.

| Atributo               | Fórmula Base         |
| ---------------------- | -------------------- |
| Vida Máxima            | 10 + Vitalidade x 2  |
| Dano Base              | Força + 1d6          |
| Defesa                 | Agilidade + Armadura |
| Chance Crítica         | Sorte x 2%           |
| Resistência Espiritual | Espírito             |

# SISTEMA ESPIRITUAL AMAZÔNICO (Diferencial)

## Favor da Floresta

Representa equilíbrio com a natureza.

* Alta → bônus espirituais
* Médio → ordem normal
* Baixa → eventos negativos

## Corrupção

Acumula ao:

* Caçar excessivamente
* Matar espíritos
* Tomar decisões egoístas

Efeito:

* Inimigos mais fortes
* Eventos sombrios
* Possível transformação do personagem

## Sanidade

Passar muitos dias na floresta:

* Pode gerar alucinações
* Eventos místicos
* Encontros com entidades

# Sistema de Progressão

A cada nível:

* +2 pontos para distribuir
* Ou escolha entre:

  * +1 atributo
  * +1 habilidade espiritual
  * Reduzir corrupção
